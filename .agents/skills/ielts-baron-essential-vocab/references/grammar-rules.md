# Grammar Rules

## Responsibilities
- Identify dominant grammar points in the reading passage
- Explain formula, nature, translation thinking (Vietnamese perspective)
- Apply to IELTS Writing Task 2
- Create ≥3 Tips & Tricks
- Analyze word class traps, agent vs concept, paraphrasing traps

## Grammar Point Structure
Each grammar point must include:
1. **Name** — Grammar structure name
2. **Formula** — Pattern/template
3. **Nature** — Why this structure exists, what it communicates
4. **Source Example** — Actual sentence from reading with source_ref
5. **Translation Thinking** — How a Vietnamese learner might approach this
6. **Writing Task 2 Application** — How to use in IELTS Writing
7. **Traps** — Common mistakes, word class confusion

## Learning Links (exactly 3 groups)
1. **Cambridge Dictionary** — Grammar reference pages
2. **British Council** — LearnEnglish grammar pages
3. **Perfect English Grammar** — Practice exercises

Link rules:
- Use verified top-level or well-known paths only
- No deep URLs that cannot be verified
- No fabricated URLs

## Tips & Tricks
- Minimum 3 tips
- Each must be actionable and specific
- Each must relate to the grammar point
- Include at least one "trap" warning
