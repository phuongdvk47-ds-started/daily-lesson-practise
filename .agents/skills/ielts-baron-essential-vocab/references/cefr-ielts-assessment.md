# CEFR & IELTS Assessment Methodology

## Assessment Dimensions
1. **Lexical Complexity** — AWL coverage, type-token ratio, word frequency bands
2. **Grammar Complexity** — Clause density, subordination, passive voice frequency
3. **Discourse Complexity** — Cohesion devices, argument structure, text organization
4. **Inference Burden** — How much implicit reasoning is required
5. **Question Difficulty** — Question type distribution and cognitive demands
6. **Academic Density** — Technical term ratio, domain-specific vocabulary

## CEFR-IELTS Mapping
| CEFR | IELTS Band Range | Description |
|------|-----------------|-------------|
| A2 | 3.0-3.5 | Elementary |
| B1 | 4.0-5.0 | Intermediate |
| B2 | 5.5-6.5 | Upper-Intermediate |
| C1 | 7.0-8.0 | Advanced |
| C2 | 8.5-9.0 | Proficiency |

## Evidence Requirements
Assessment must provide specific evidence for each dimension, not just word count or passage length.

## Confidence Thresholds
- ≥ 0.85: Auto-approve
- 0.75-0.84: Auto-approve with note
- < 0.75: Human review required

## Human Override
Users may override assessment with reason. Original assessment is preserved.
