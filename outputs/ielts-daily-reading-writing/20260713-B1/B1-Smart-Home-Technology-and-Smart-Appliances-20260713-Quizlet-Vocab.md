### Phần 1: Học từ vựng đơn giản (Simple Vocabulary)
| Từ vựng tiếng Anh | Nghĩa tiếng Việt |
| --- | --- |
| integrated | Combining many parts into a single whole |
| interconnected | Connected with each other |
| remote management | The control of a system from a distant location |
| unprecedented | Never having happened or existed in the past |
| automated routine | A series of actions performed by systems without human help |
| energy efficiency | Using less energy to perform the same task |
| occupancy sensor | A device that detects if a room is being used |
| consumption | The amount of something that is used |
| environmental footprint | The effect that human activities have on the environment |
| interoperability | The ability of different systems to work together |
| non-technical | Not relating to science or technology |
| learning curve | The rate of someone's progress in learning a new skill |
| widespread adoption | The act of accepting or using something very widely |
| behavioral data | Information collected about people's activities and habits |
| vulnerable | Easy to hurt, influence, or attack |
| cyberthreat | A threat of hacker attacks on computers or networks |
| default password | A password set by the manufacturer for initial setup |
| device firmware | Software program written into a hardware device |
| smart appliance | A household device connected to the internet |
| seamlessly | Smoothly, without any gaps or problems |
