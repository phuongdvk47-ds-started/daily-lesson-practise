### Phần 1: Học từ vựng đơn giản (Simple Vocabulary)
| Từ vựng tiếng Anh | Nghĩa tiếng Việt |
| --- | --- |
| habit | A thing that you do often and almost without thinking |
| routine | The normal order and way in which you regularly do things |
| schedule | A plan that lists all the work that you have to do |
| productivity | The rate at which a person does useful work |
| focus | The concentration of attention on a particular activity |
| active | Doing things that require physical effort and energy |
| alert | Able to think quickly and notice things |
| lifestyle | The way in which a person lives |
| commute | To travel regularly between home and work or school |
| healthy | Helping to produce good health |
| timetable | A list showing the times at which events will happen |
| prioritize | To treat something as being more important than other things |
| wellbeing | The state of being comfortable, healthy, or happy |
| distraction | Something that takes your attention away from study |
| facilities | Buildings, services, or equipment provided for a purpose |
| working memory | Memory used to hold information temporarily |
| musical training | Learning to play music or an instrument |
| cognitive | Connected with mental processes of learning |
| time management | The ability to use one's time effectively |
| on track | Progressing in the way that was planned |
